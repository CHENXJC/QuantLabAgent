## Screenshot Rules

- Avoid showing API keys, tokens, personal data, or private local paths.
- Keep browser zoom around 90% to 100%.
- Prefer clean screenshots with sidebar visible.
- Use Chinese screenshots as the main display.
- Use English screenshots as supporting proof of bilingual interface.
- Do not present outputs as investment advice.
